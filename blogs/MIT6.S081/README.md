## 项目介绍
Mit6.s081是麻省理工学院面向本科生的操作系统课程，其课程实验是在教学用操作系统xv6上进行扩展和优化。
Xv6操作系统源代码只有一万余行，并且相对清晰和模块化，每个lab都带有自动评测功能，非常适合用于实践操作系统知识。
